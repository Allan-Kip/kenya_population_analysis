#Kenya Population Analysis (2019 Census)

**Objectives**

-Analyze population distribution by county.
-Identify top and least populated regions.
-Visualize overall population patterns.
-Provide insights for planning and development.

**Tools Used**

-tidyverse, ggplot2, readr, scales

**Outputs**
-top10_counties.png
-bottom10_counties.png
-population_distribution.png
-population_share_pie.png
-population_summary.csv

**Conclusion**
Kenya’s 2019 Census reveals large differences in county populations — urban centers like Nairobi, Kiambu, Nakuru, and Kakamega lead, while northern and coastal counties remain sparsely populated.
This uneven distribution emphasizes the need for targeted resource allocation and balanced regional development.
